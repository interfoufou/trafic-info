## 🚀 النشر على Netlify

### الطريقة الأولى: من GitHub (موصى بها)

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start)

#### الخطوات:

1. **إنشاء مستودع GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/trafic-info-tunisie.git
   git push -u origin main
   ```

2. **ربط Netlify:**
   - اذهب إلى https://app.netlify.com
   - اضغط **"Add new site"** → **"Import an existing project"**
   - اختر **"GitHub"**
   - اختر مستودع `trafic-info-tunisie`
   - اضغط **"Deploy site"**

### الطريقة الثانية: السحب والإفلات

1. قم بتحميل ملف ZIP من [هنا](/trafic-info-tunisie-netlify.zip)
2. اذهب إلى https://app.netlify.com/drop
3. اسحب مجلد المشروع
